# AI 写作助手 Lite

轻量级写作助手，无需外部服务，自然对话式交互。

## 执行流程

当用户请求写作时：

1. **初始化**（首次运行）
   - 创建数据库和目录结构
   - 检查依赖（jieba 可选）

2. **理解意图**
   - 自动识别输入类型（文本/文件/JSON）
   - 长文本（>50字）自动导入为素材

3. **自然对话引导**
   ```
   AI: "好的！您有相关的参考材料吗？可以直接粘贴，或者我直接创作。"
   ```

4. **生成文本**
   - 有素材：生成带引用的文本
   - 无素材：直接创作
   - 保存到临时文件供编辑

5. **等待修改**
   ```
   AI: "初稿已完成，保存在 draft_xxx.md。您可以在编辑器中修改，改好后告诉我。"
   ```

6. **处理修改**
   - 计算三粒度 diff
   - 检查是否触发萃取（3条或10条）
   - 自动更新偏好模型
   - 清理临时文件

## 注意事项

- API Key 可选：未配置时跳过自动萃取
- jieba 可选：未安装时仅提示
- 临时文件自动清理：24小时后删除
